## Instalation

1. **Clone The Repositories:**
```bash
git clone https://github.com/airdropbomb/pspiAutoBot.git && cd pspiAutoBot
```

2. **Install Requirements:**
   ```bash
   npm install
   ```
3. **Enter Your Private Key**
   ```bash
   nano pk.txt
   ```
4. **If You Want use proxy enter your proxy**
   ```bash
   nano proxy.txt
   ```
**Run Bot**
   ```bash
   npm start
   ```
